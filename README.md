## 학습 
### 1. CSS : animation의 step 옵션 
: 프레임을 설정해 주는 기능입니다.  
: 애니메이션 옵션에서 기본 설정은 자연스럽게 움직이는 것이지만 steps옵션을 넣어주면 몇 단계를 통해서 애니메이션을 보여줄 것인지 설정할 수 있습니다.
```
  animation: typing 2s steps(22) infinite; 
  // 2초에 걸쳐서 22단계를 걸쳐서 typing을 실행  
```
